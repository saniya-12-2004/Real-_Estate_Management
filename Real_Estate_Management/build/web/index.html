About us

Property dealing is user friendly web application developed in java programming language as front end and My SQL server database as back end. This project is designed to save the data of all those person who will hire, lease or buy any kind of property like buildings, flats, plots, etc. which will reduce manual work helps the dealer to save the records efficiently. This project aims to develop a comprehensive real estate management system for residential and commercial properties, focusing on enhancing property value tenant satisfaction and financial performs. In Old days, the Property Dealer may have to keep records of the properties manually. This method of keeping the records is quite time consuming and less efficient. Also there are more chances of mistakes by keeping the records the manually as human beings are habitual of doing mistakes .So with the help of this, property dealing web application. It consumes very less time as compared to manual method. The application is divided into three modules. The Admin i.e. the Property Dealer itself handles the whole application. He managed the database as well as all the records.
